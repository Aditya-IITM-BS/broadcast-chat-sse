# Broadcast Using Server Sent Events

### clone using 
`$git clone `

### create virtual environment
windows:
`$python -m venv .venv`
linux
`$python3 -m venv .venv`

### activate virtual environment
windows:
`$.\.venv\Script\activate`
linux
`$source .venv/bin/activate`

### install dependencies
`$pip install -r requirements.txt`

### run
`$python run app.py`
linux
`$python3 run app.py`
